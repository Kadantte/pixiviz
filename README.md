<h1 align="center">Pixiv C</h1>

## 概述

尝试使用 [Pixivic.com](https://pixivic.com) 的接口，做一个具有清新画风，支持一定程度响应式的前端。

项目使用 Vue-cli 建立，配合 Element UI 编写。

## 开发目标

- [x] 完成Landing界面
- [x] 完成排行榜界面
- [ ] 完成画作详情界面
- [ ] 完成搜索界面

## 使用方式

安装依赖：
```
npm install
```

运行本地服务器，实时编译 & 热重载：
```
npm run serve
```

推荐使用 Vue-cli 的 GUI 来管理本项目。
